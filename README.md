# Scorekeeper App

A simple, beautiful scorekeeper for up to 3 players (or more!).

## How to Start

1. **Open your terminal** and navigate to this folder.

2. **Ensure the start script is executable** (you only need to do this once):
   ```bash
   chmod +x start.sh
   ```

3. **Run the start script**:
   ```bash
   ./start.sh
   ```

4. **Open your browser** and visit:
   [http://localhost:3000](http://localhost:3000)
